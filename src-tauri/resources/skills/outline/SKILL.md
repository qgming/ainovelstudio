# 大纲规划 Skill

统一负责构思期与大纲期,把故事从“概念”推进到“可写的大纲”。

## 阶段划分

| 阶段 | 内容 | 输出物 |
|------|------|--------|
| 构思期 | 一句话概括 + 五句式大纲 | `00-一句话概括.md` `01-五句式大纲.md` |
| 扩展期 | 一页纸大纲 | `02-一页纸大纲.md` |
| 完整期 | 完整大纲 + 一致性检查 | `05-完整大纲.md` |

## 使用原则

- 如果工作区只有创意或还没有任何大纲文件,从构思期开始。
- 如果已有 `00-一句话概括.md` 与 `01-五句式大纲.md`,直接进入扩展期。
- 如果已有 `02-一页纸大纲.md`,默认直接进入完整期。
- 用户带 `--rebuild` 时,从头重建所有阶段。

## 构思期要点

- 先产生一句话概括,明确主角、困境、行动与潜在结局暗示。
- 再扩展为五句式大纲,形成清晰的三幕结构与三次灾难性事件。
- 可同时生成 `00-写作风格.md` 作为后续正文生成底座。

## 扩展期要点

- 将五句式扩展为约 400-600 字的一页纸大纲。
- 各段需包含具体场景、关键对话与情绪氛围。
- 保持因果链清晰,不要写成五段互不关联的摘要。

## 完整期要点

- 结合人物卡、人物背景与时间线类型生成 2000-3000 字的完整大纲。
- 每章至少包含: 章节标题、场景设定、POV、核心事件、情感锚点、钩子。
- 额外附上一致性检查结果,覆盖人物、时间线、三幕结构与伏笔。

## 与其他技能配合

```
outline
   ↓
character-design  ← 补齐人物卡、背景与人物宝典
   ↓
scene-plan        ← 拆成场景清单与逐场规划
   ↓
chapter-write     ← 进入正文生成
```
