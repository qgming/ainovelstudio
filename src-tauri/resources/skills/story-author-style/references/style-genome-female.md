# 女频作者风格基因库

每个基因是结构 / 节奏 / 句法层面的**可执行指标**，不是抄桥段、设定专名。套用时以项目作者既有文风为准，基因只补不覆盖。

风格判断综合自读者社区（知乎、豆瓣）的公开讨论与媒体评述，带主观性，作为创作参考而非定论。

---

## 顾漫型 · 温馨轻快、细节见长、甜宠

**一句话定位**：温馨轻快的日漫式甜，靠细腻细节调动共鸣，霸总宠溺，节奏舒缓。

| 维度 | 指标 |
|---|---|
| 句长分布 | 中短句为主，轻快不拖；情绪点用短句轻轻一顿。 |
| 标点预算 | 规整；逗号自然，少用感叹号，靠语气而非标点煽情。 |
| 动名词比 | 细节名词丰富（小动作、物件、表情），动词轻巧。 |
| 对话密度 | 高；对话甜、有来有回，靠互动推进关系。 |
| 段落谱系 | 段落短、节奏匀，日常场景铺细节。 |
| 情绪外化 | 用细节和微动作外化心动 / 别扭 / 甜，不直喊「我爱你」。 |
| 章末钩子 | 柔钩——留一个甜的悬念或关系小进展，牵引追读。 |

**标志性技法**
- 细节调情：用生动具体的小细节（一个动作、一句话、一个物件）放大心动和张力。
- 反差宠溺男主：表面疏离 / 傲慢，对女主极致护短宠溺，反差出萌点。
- 轻喜剧节奏：日常甜 + 小误会 + 化解，圆满走向。

**慎用项**
- 舒缓节奏在强爽点免费平台慎用，需提速、前置冲突。
- 甜不等于无冲突，仍要有情感阻力和关系张力。

## Priest 型 · 精练大气、剧情驱动、立主旨

**一句话定位**：精练幽默有门槛，人物服务剧情，作品有明确主旨内核。

| 维度 | 指标 |
|---|---|
| 句长分布 | 长短交错，长句承载信息与机锋，短句收束。 |
| 标点预算 | 克制规整；幽默靠句子结构而非标点。 |
| 动名词比 | 动作、场景、对话穿插推进；少大段静态心理。 |
| 对话密度 | 高；对话精妙、带幽默和潜台词，是推进主力。 |
| 段落谱系 | 紧凑，按剧情节拍分段，几乎不写成段长心理。 |
| 情绪外化 | 情绪藏在动作和对话里，靠剧情处境带出，不直抒。 |
| 章末钩子 | 剧情钩为主，靠事件推进牵引。 |

**标志性技法**
- 人物服务剧情：人物动机清晰，几乎不写大段心理独白，靠动作 / 场景 / 对话走剧情。
- 立主旨：故事有一个站得住的核心命题（家国 / 公平 / 自由），三观正。
- 精妙比喻 + 幽默：用比喻和冷幽默对话提升质感和推进力。

**慎用项**
- 文笔有阅读门槛：复杂多变。面向零门槛快节奏读者时需降低门槛、提清晰度。
- 主旨先行别说教：主旨要藏在剧情里，不靠角色喊口号。

## 墨香型 · 流畅零门槛、副本短篇式、重人物

**一句话定位**：语言流畅零门槛，副本式短篇串联，节奏紧凑，重人物塑造。

| 维度 | 指标 |
|---|---|
| 句长分布 | 中短句为主，简洁明快，读着不费力。 |
| 标点预算 | 简洁规整；不堆复杂句式。 |
| 动名词比 | 动作清楚、语言直白；情绪靠简洁有力的句子带。 |
| 对话密度 | 高；对话生动直白，塑造角色性格。 |
| 段落谱系 | 紧凑，副本 / 单元各自完整，像一串短篇。 |
| 情绪外化 | 心理描写细腻但简洁，凸显主角特质与英雄主义。 |
| 章末钩子 | 单元收尾平稳，靠人物魅力和下一副本悬念续读。 |

**标志性技法**
- 副本短篇式：把每个单元当短篇写，结构完整、推进快、不注水。
- 重人物英雄主义：情节围绕凸显主角特质展开，人物鲜活先于设定复杂。
- 零门槛易读：用最易懂的语言让读者快速沉浸。

**标志性技法的迁移价值**：副本短篇式结构非常适合短篇赛道和单元连载——每个单元自成爽点闭环。

**慎用项**
- 短篇思维拉长篇易「质地疏松」：写长篇时要补长线主线，别只靠单元串联。

## 番茄 / 盐言短篇型 · 接地气、爽点密集、反转打脸

**一句话定位**：免费平台 / 短篇向的强情绪流，节奏极快，爽点密集，反转打脸。

| 维度 | 指标 |
|---|---|
| 句长分布 | 短句为主，强发力；段落推进快。 |
| 标点预算 | 偏口语；感叹号 / 问号制造情绪冲击但不过度。 |
| 动名词比 | 动作直给，情绪外放，拒绝静态铺陈。 |
| 对话密度 | 高；冲突对话密集，撕、怼、打脸靠对话。 |
| 段落谱系 | 短段为主，独句成段制造冲击，留白快切。 |
| 情绪外化 | 情绪强外放，矛盾冲突直给，前 3 万字（或短篇前段）拒绝场景细节铺陈。 |
| 章末钩子 | 强钩——每段抛悬念、埋反转、设打脸预期，环环相扣。 |

**标志性技法**
- 开篇即冲突：第一段就甩矛盾 / 关系裂缝，强情感渲染立人设，拉高期待。
- 套路 + 反转：先用熟悉套路建立预期，再快速反转打脸，短篇尤其要快建快翻。
- 雷点细分：精确瞄准目标读者的爽点 / 痒点（重生复仇、替身、追妻火葬场、萌宝、医妃等）。
- 持续悬念链：解开一个悬念同时抛出更大悬念，环环相扣维持代入感。

**女频爽点逻辑**（与男频不同）
- 男频「为变强不顾一切」；女频常是「强大男主为爱女主不顾一切」+「前期被虐、后期地位反转回虐」。
- 甜文 / 虐文两极：要么使劲撒糖，要么虐身虐心，情绪给到位。

**慎用项**
- 节奏极快不等于没逻辑：反转仍需前置至少 2 个可回看线索，否则垮。
- 强情绪别变狗血：人设动机要立得住。
