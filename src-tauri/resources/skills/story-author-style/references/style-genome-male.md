# 男频作者风格基因库

每个基因是结构 / 节奏 / 句法层面的**可执行指标**，不是抄桥段、设定专名。套用时以项目作者既有文风为准，基因只补不覆盖。

风格判断综合自读者社区（知乎、豆瓣）的公开讨论与媒体评述，带主观性，作为创作参考而非定论。

---

## 辰东型 · 开篇即奇观、宏大气势

**一句话定位**：开局甩出一个压不住的奇观或谜团，靠尺度和悬念抓人，热血外放。

| 维度 | 指标 |
|---|---|
| 句长分布 | 中长句铺气势 + 关键处短句砸点；造势段落句子偏长。 |
| 标点预算 | 逗号多用于铺排递进；感叹号在热血点放，但不刷屏。 |
| 动名词比 | 大量具体名词堆尺度（山河、星空、古棺、神祇），动词重「镇、压、轰、撕」。 |
| 对话密度 | 偏低，叙事 / 描写为主；对话多为放狠话、立目标。 |
| 段落谱系 | 造势段长，揭谜 / 转折处独句成段制造停顿。 |
| 情绪外化 | 靠场面尺度和旁观者反应外化，不直写「他很震撼」。 |
| 章末钩子 | 强钩——抛更大的谜或更强的敌，下一章必须接得住。 |

**标志性技法**
- 开篇奇观法：第一段就给一个反常 / 宏大 / 诡异的画面（如「九龙拉棺」式开局），先抓眼再解释。
- 悬念高密度：埋问题的速度快于解答，始终欠读者一个答案。
- 尺度递进：每个阶段把世界观尺度往上抬一级，制造「原来上面还有」的压迫感。

**慎用项**
- 中后期易注水：副本来回刷、尺度通胀导致疲劳。套用时控制奇观频率，给爽点兑现节点。
- 对话少 + 句子长，节奏偏重；快节奏短篇 / 番茄向慎用，需缩句提密度。

## 乌贼型 · 设定先行、局中有局

**一句话定位**：靠精巧设定和环环相扣的信息结构取胜，文笔服务于逻辑。

| 维度 | 指标 |
|---|---|
| 句长分布 | 平实、信息密度高，长句用于交代设定与因果。 |
| 标点预算 | 规整克制；少炫技标点。 |
| 动名词比 | 名词多为设定术语（体系、序列、仪式），动词朴实，重在说清机制。 |
| 对话密度 | 中等；对话常用于释放设定信息和推进谜题。 |
| 段落谱系 | 段落组织严整，按信息单元分段，逻辑链清晰。 |
| 情绪外化 | 情绪让位于剧情，靠处境变化带动，不大段抒情。 |
| 章末钩子 | 用「解开一个谜同时抛出更大谜」的信息钩，一波三折。 |

**标志性技法**
- 设定即卖点：世界观规则自洽且新奇，规则本身制造爽点和悬念。
- 局中有局：表层事件下藏更深布局，信息分层释放，回看处处有伏笔。
- 长线收束：埋的设定 / 伏笔最终成体系回收，适合长篇。

**慎用项**
- 文笔评价偏「一般」：不要靠华丽辞藻，要靠设定密度和结构取胜。
- 设定门槛高：开篇信息别一次塞太满，需用钩子带读者逐步进入。

## 烽火型 · 散文笔触、辞藻唯美、人物细腻

**一句话定位**：文笔为先，细腻唯美，人物心理与江湖气浓，慢热。

| 维度 | 指标 |
|---|---|
| 句长分布 | 长句为主，讲究韵律与意象；短句用于点睛。 |
| 标点预算 | 逗号绵密营造绵长语感；可用顿挫但不靠感叹号发力。 |
| 动名词比 | 形容词 / 意象密度高于一般网文，但落在具体物象上而非空泛大词。 |
| 对话密度 | 中高；对话有机锋、留白、半句，人物声音区分度高。 |
| 段落谱系 | 长短段交替，写景 / 抒情段偏长，对话段轻快。 |
| 情绪外化 | 心理描写细腻，但靠场景与动作承载，不直白喊情绪。 |
| 章末钩子 | 偏情绪 / 意境收尾，钩子柔，靠人物牵引续读。 |

**标志性技法**
- 意象造境：用具体物象（刀、雪、酒、江湖）堆出氛围与人物气质。
- 人物声音：每个角色台词有辨识度，机锋和留白见功力。
- 闲笔养人物：用看似无用的日常笔墨把人物写丰满。

**慎用项**
- 慢热、偏散文化，易被指「注水」。短篇 / 强爽点快节奏赛道慎用，或只取其句法不取其节奏。
- 辞藻唯美不等于堆 AI 大词；意象必须落到具体物象，否则变空泛。

## 唐三型 · 干净纯净、架空想象、规律爽点

**一句话定位**：文字干净纯净、低龄友好，架空想象 + 稳定规律的升级爽点。

| 维度 | 指标 |
|---|---|
| 句长分布 | 中短句为主，读着顺、门槛低。 |
| 标点预算 | 规整简洁；不炫技。 |
| 动名词比 | 动作清楚，形容词适度；不堆复杂修辞。 |
| 对话密度 | 中等；对话直白易懂，推动关系和升级。 |
| 段落谱系 | 段落短、节奏匀，读起来轻快。 |
| 情绪外化 | 情绪表达直接、正向，少阴暗复杂。 |
| 章末钩子 | 稳定埋下一个升级 / 比试 / 危机点，节奏规律。 |

**标志性技法**
- 干净尺度：杜绝色情 / 血腥 / 暴力泛滥，情爱发乎情止乎礼，适配低龄与泛读者。
- 架空造世：脱离现实的全新世界（大陆 / 神界 / 未来），想象自由。
- 规律爽点：升级 / 兑现节奏稳定可预期，养读者追读习惯。

**慎用项**
- 文字「纯净」不等于平淡无冲突；仍需明确目标、冲突和爽点。
- 低龄友好≠低幼，避免幼稚化人物动机。
