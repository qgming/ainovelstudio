# 时间线设定参考

## 用途

本文档提供完整的时间线模板，适合管理历史背景、主线推进、人物阶段变化和伏笔回收。
它覆盖两个层级：

- 简版时间线卡：适合快速确认主线顺序和核心节点
- 完整时间线手册：适合长篇、连载、多线并行和复杂因果结构

## 使用建议

- 短篇：优先锁定触发事件、转折点和结局回收
- 中篇：补齐主线事件和人物阶段变化
- 长篇 / 连载：建立卷级节点、节奏单位和阶段性 payoff
- 悬疑、多线叙事、无限流：重点填写信息揭露顺序和回收时点

## 时间线检查清单

- [ ] 角色能在合理时间到达对应地点
- [ ] 事件顺序支持人物动机变化
- [ ] 伤势、能力、资源和关系变化留出恢复时间
- [ ] 伏笔在合理时间点回收
- [ ] 信息揭露顺序和剧情推进互相配合
- [ ] 长篇具备卷级推进感
- [ ] 短篇没有铺垫超过回收长度
