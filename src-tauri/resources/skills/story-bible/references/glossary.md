# 术语设定参考

## 用途

本文档提供完整的术语管理模板，适合统一专有名词、制度名称、能力名称、称谓和固定表达。
它覆盖两个层级：

- 简版术语卡：适合快速锁定一个核心术语
- 完整术语手册：适合复杂题材中的命名体系、术语层级和解释策略

## 使用建议

- 短篇：优先保留最少量、最易理解的关键词
- 中篇：补齐主术语、别名和使用规范
- 长篇 / 连载：建立核心术语与扩展术语的层级关系
- 科幻、玄幻、仙侠、规则类题材：重点填写首次解释策略、口语 / 正式写法和读者理解成本
