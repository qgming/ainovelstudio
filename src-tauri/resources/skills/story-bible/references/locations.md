# 地点设定参考

## 用途

本文档提供国家、城市、区域、建筑和关键场景的地点设定模板。
它覆盖两个层级：

- 简版地点卡：适合快速锁定场景功能与记忆点
- 完整地点手册：适合主舞台、反复回收场景和长篇地理系统

## 使用建议

- 短篇：先写地点记忆锚点、冲突价值和情绪价值
- 中篇：补齐空间结构、移动成本和已发生事件
- 长篇 / 连载：建立地点层级、路线逻辑和重复使用方式
- 悬疑、无限流、末世、玄幻、仙侠：重点填写环境限制、隐藏区域和场景触发器

## 地点设定检查清单

- [ ] 地点具备独立记忆点
- [ ] 地点不只是背景板，而是能推动剧情
- [ ] 移动成本符合时间线
- [ ] 隐藏区域和公开区域逻辑自洽
- [ ] 长篇地点有复用价值
- [ ] 短篇地点数量不过载
- [ ] 环境描写浓度符合平台节奏
