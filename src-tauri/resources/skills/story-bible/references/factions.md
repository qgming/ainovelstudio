# 阵营与组织设定参考

## 用途

本文档提供阵营、组织、家族、公司、宗门、帮派等群体设定模板。
它覆盖两个层级：

- 简版组织卡：适合快速确认冲突阵面
- 完整组织档案：适合长期连载中的势力网络、权力斗争和资源体系

## 使用建议

- 短篇：优先写组织目标、对主角态度和当前冲突功能
- 中篇：补齐资源、内部矛盾、对外关系
- 长篇 / 连载：建立势力层级、阶段敌友变化和升级空间
- 权谋、基建、商战、帮派、宗门题材：重点写组织文化与权力运转方式

## 组织设定检查清单

- [ ] 组织目标与行动方式匹配
- [ ] 组织资源能支撑其地位
- [ ] 内部矛盾真实存在
- [ ] 对主角的态度能形成持续戏剧张力
- [ ] 敌友关系不是静态摆设
- [ ] 长篇具备势力升级链
- [ ] 短篇没有让组织复杂度压过剧情本身
