# 世界设定参考

## 用途

本文档提供完整的世界设定模板，适合 `story-bible` 在建档、扩写和更新世界观 canon 时使用。
它覆盖两个层级：

- 简版世界卡：适合快速锁定故事世界的骨架
- 完整世界观宝典：适合主世界、主要副本、核心体系和长期连载设定

## 使用建议

- 短篇：先写简版世界卡，再补最关键的规则与冲突来源
- 中篇：补齐世界规则、社会秩序和剧情接口
- 长篇 / 连载：优先建立完整世界观宝典，尤其是规则边界、势力结构和长期威胁
- 科幻、玄幻、仙侠、无限流等高设定题材：重点填写力量体系、资源系统和世界运行逻辑

## 世界设定检查清单

- [ ] 核心规则能直接影响剧情
- [ ] 力量或科技体系有明确边界和代价
- [ ] 社会结构和世界规则彼此匹配
- [ ] 文化习俗服务人物选择和冲突
- [ ] 开篇设定投放量可控
- [ ] 长篇设定具备持续扩展空间
- [ ] 短篇设定没有超过剧情承载力
