# 人物设定参考

## 用途

本文档提供完整的人物设定模板，适合 `story-bible` 在建档、扩写和更新人物 canon 时使用。
它覆盖两个层级：

- 简版人物卡：适合快速建立角色基础设定
- 完整人物宝典：适合主角、核心配角、反派和 CP 关系的深入挖掘

## 使用建议

- 主角：优先填写简版人物卡，再补完整人物宝典
- 反派：重点填写深层心理、背景故事、行为底层逻辑和记忆点
- 配角：使用简版人物卡，再补 1 到 2 个强记忆点
- 工具人：仅填写简版中的核心设定和一句话经历
- 言情或多角色关系文：补充 CP 关系设计和社交关系

## 人物一致性检查清单

- [ ] 对话风格符合人物设定
- [ ] 行为符合价值观
- [ ] 行为遵循角色的底层行为逻辑
- [ ] 情绪反应符合性格
- [ ] 标志性语言或动作得到稳定使用
- [ ] 与其他人物的关系保持一致
- [ ] 角色反常行为有充分铺垫
- [ ] 配角行为基于立场，而不是简单善恶
