# 情节检查模板

用于检查章节内容或最近一次大纲变更里的事件推进、因果链、题材匹配和结构设计是否成立。

## 1. 章节推进与因果链检测

核心问题：章节发生了很多事，但没有有效推进，或者推进依赖巧合和硬转折。

### 检查清单

- 本章主目标是否清楚
- 本章内容是否服务章节目标、卷级推进或角色弧线
- 事件之间是否有明确因果
- 结果是否来自人物选择、环境压力或既有规则
- 转折是否有前置条件或信息铺垫
- 章末钩子是否建立在本章真实推进上

### 常见问题信号

- 情节靠巧合、旁白解释或外力硬推
- 写了很多内容，但局面没有变化
- 转折看似刺激，前文没有必要支撑
- 章末钩子和本章核心冲突关系很弱

## 2. 大纲更新后的题材与内容匹配检测

核心问题：标签、题材方向和实际内容不匹配，导致读者预期落空。

### 检查清单

- 主标签与主线是否匹配
- 主题材是否占主要比重
- 副题材是否清楚服务主题材
- 大纲更新后，读者冲着标签来的预期是否仍能被满足

### 常见问题信号

- 写的是玄幻，主线却长期停在都市生活
- 混入新元素后，主线不再清晰
- 标签没有变，内容方向已经变了

## 3. 辨识度检测

核心问题：更新后的大纲和剧情过于常见，缺少记忆点和选择理由。

### 检查清单

- 一句话能否说清这本书独特在哪里
- 更新后的主线和同类作品相比，有没有清晰差异
- 读者能记住哪一个独特元素
- 新增设定是否增强了辨识度，而不是稀释了卖点

### 常见问题信号

- premise 改完之后，像任何一本常规系统文或重生文
- 加了很多新设定，却没有一个能成为记忆钩子
- 独特设定存在，但不影响主线

## 4. 跨题材混搭检测

核心问题：混搭元素越来越多，最后变成四不像。

### 检查清单

- 混搭元素数量是否仍在可控范围
- 主题材和副题材是否主次分明
- 元素之间是否能发生化学反应
- 读者是否还能快速理解这是什么类型的书

### 常见问题信号

- 都市、修仙、系统、末世、重生同时堆上来
- 新增元素只是挂件，对主线没有贡献
- 各元素都沾一点，但没有任何一条线足够强

### 修复提示

- 保留主题材和最强的 1 到 2 个副元素
- 删除只增复杂度、不增卖点的混搭项

## 5. 世界观设定呈现检测

核心问题：设定过多、过早或过散，读者记不住，也无法转化成剧情张力。

### 检查清单

- 当前阶段需要读者记住的核心概念是否过多
- 设定是否通过场景、对话和行动展开
- 新设定是否真的服务当前剧情
- 更新后的世界规则是否和既有正文、自有体系保持一致

### 常见问题信号

- 开篇或新卷前几章连续灌设定
- 规则一改，前文已有事件解释不通
- 设定很复杂，但人物没有因此做出新选择

### 修复提示

- 只保留当前剧情必须知道的设定
- 让设定通过冲突和选择显现，而不是纯说明

## 输出建议

优先标出：

- 因果断点
- 推进不足
- 转折失真
- 题材偏移
- 卖点稀释
- 设定倾倒或规则冲突
