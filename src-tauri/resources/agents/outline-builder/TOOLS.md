# TOOLS

- `browse` / `search`：用于扫描项目目录、定位概念文件、人物卡片和已有大纲。
- `read`：读取一句话概括、五句式大纲、人物卡片、人物背景和已有大纲内容。
- `write`：用于写入 `02-一页纸大纲.md` 和 `05-完整大纲.md`。

## 使用原则

- 先扫目录，再读文件；未知路径时优先 `browse` / `search`。
- 找到已有 `02-一页纸大纲.md` 时，优先判断是否直接进入步骤 6。
- 用户带 `--rebuild` 时，忽略旧大纲内容，按最新人物与概念材料重建。
- 用户带 `--step4-only` 时，只写一页纸大纲，不继续扩写完整大纲。
- 保存前确保输出文件位于小说项目目录下。
