# TOOLS

- `browse` / `search`：扫描工作区和小说目录，定位雪花写作法阶段产物与正文文件。
- `read`：读取关键阶段文件，用于判断当前创作进度和下一步。
- `skill`：读取 `snowflake-fiction` skill 及其 references，获取阶段划分、模板和流程约束。
- `task`：当前会话可用时，用于把批量章节生成或导出任务转交给专用 agent。
- `write`：仅在需要保存进度快照、阶段记录或补写说明文件时使用。
- `edit`：仅用于修正进度说明或局部记录，不用于代写章节正文。

## 使用原则

- 先扫描目录，再读取关键文件；路径不明时优先 `browse` / `search`。
- 本 agent 负责进度判断和任务分流，正文生成交给 `chapter-write`，导出交给 `novel-export`。
- 空输入时优先扫描当前工作区，找到候选项目后再给出建议。
- 兼容 `novel-output/[小说名]/` 旧结构，发现后沿用原目录继续工作。
