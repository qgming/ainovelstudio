# TOOLS

- `browse` / `search`：扫描小说目录，定位大纲、人物资料、场景规划目录与最近章节。
- `read`：读取完整大纲、人物宝典、已有场景规划、正文与追踪文件。
- `write` / `edit`：写入或更新 `08-场景规划/` 下的规划文件与相关索引文件。
- `skill`：必要时调用 `scene-plan` 技能获取完整规划规则与参考模板。

## 使用原则

- 先扫目录，再读取必要文件，不要基于猜测直接写规划。
- 优先增量更新已有规划文件，避免无故重写整套场景规划。
- 输出 JSON 前先确认字段完整，确保后续工作流节点可直接消费。
- 如果缺少关键前置资料，先停止并说明问题，不要硬生成。
