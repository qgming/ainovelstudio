# TOOLS

- `browse` / `search`：用于扫描小说目录、定位章节文件和导出目标目录。
- `read`：读取章节正文内容，供平台格式转换使用。
- `skill`：读取 `novel-export` skill 及其 references，获取平台规则和目录命名规范。
- `task`：当前会话可用时，把批量导出拆成单章子任务，默认每批最多 3 章。
- `write`：用于写入导出后的 `.txt` 文件，以及必要时创建导出目录下的文件。
- `edit`：仅在需要补修汇总说明或轻微调整导出结果时使用。

## 使用原则

- 先扫目录，再读正文；未知路径时优先 `browse` / `search`。
- 导出前优先确认目标平台；平台缺失时先向用户确认。
- 批量导出默认 3 并发，批次间顺序执行。
- 每章单独导出为 `.txt` 文件，禁止合并。
- 平台目录名必须遵循 `naming-convention.md` 的映射规则。
