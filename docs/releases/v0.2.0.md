# 神笔写作 v0.2.0 更新日志

发布日期：2026-04-23

## 版本亮点

本次更新围绕「Workflow 节点交接收敛」「数据管理备份可见性」和「0.2.0 发版收口」三条主线推进。工作流节点之间的交接更聚焦，循环切换时的瞬时上下文会被主动清空；数据管理页现在会明确告诉用户完整备份范围，模型配置、主代理 AGENTS 和 WebDAV 设置都会随备份一起恢复。

## 重点更新

### 1. Workflow 节点提示词与交接链路重构

- 将步骤提示词构建逻辑从 `engine.ts` 中拆出，独立沉淀为 `stepPrompt.ts`
- 为 `agent_task` 与 `decision` 分别定义更清晰的提示词结构
- 节点提示词现在统一包含：
  - 工作流目标
  - 节点身份
  - 执行边界或判断契约
  - 当前任务
  - 交接上下文
- 判断节点继续使用 `workflow_decision` 工具作为程序分支契约，职责边界更明确

### 2. Workflow Delta Memory 去重与新循环重置优化

- `buildWorkflowDeltaMemory` 新增对上一节点结构化消息的识别能力
- 上一步已交接的消息、`review_result`、`revision_brief` 和重复消息会在增量记忆中自动去重
- “上一步增量记忆”文案调整为“上一步交接”，更贴近实际语义
- 新增 `resetRuntimeForNextLoop`
  - 进入下一轮循环时重置 `attemptIndex`
  - 清空上一轮的步骤运行缓存
  - 清空上一轮的结构化消息缓存
  - 清空上一轮的审查结论与分支状态
- 返修与循环执行现在更容易避免把上一轮瞬时上下文误带到下一轮

### 3. 数据管理页明确完整备份范围

- 数据管理页新增“完整备份范围”卡片
- 本地备份和云备份都明确展示以下内容会被一起备份：
  - 书籍工作区
  - 对话记录
  - 技能 / 代理 / 工作流
  - 模型配置
  - 工具开关
  - 主代理 AGENTS.md
  - WebDAV 配置
- 本地导入和云端下载成功提示同步强调“模型配置也会一并恢复”
- 云端下载确认弹窗同步补充“模型配置与页面偏好会被覆盖”的提醒

### 4. 版本号与更新测试同步到 0.2.0

- 项目版本号由 `0.1.9` 升级到 `0.2.0`
- `package.json`、`package-lock.json`、`Cargo.toml`、`Cargo.lock`、`tauri.conf.json` 与 README 展示版本已同步更新
- 关于页、侧边栏更新入口、更新 store 等测试样例同步切换到：
  - 当前版本 `0.2.0`
  - 可用更新样例 `0.2.1`
