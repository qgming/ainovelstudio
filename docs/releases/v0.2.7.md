# 神笔写作 v0.2.7 更新日志

发布日期：2026-05-10

## 版本亮点

本次更新从 `0.2.6` 推进到 `0.2.7`，重点修复桌面镜像同步的旧内容回灌风险，并优化 Windows 桌面端退出与托盘轻量模式体验。

## 重点更新

### 1. 镜像改动自动同步回内部工作区

- 打开系统文件资源管理器后，应用会记录当前镜像内容基线。
- 镜像文件夹中的新增、删除、重命名和内容修改会自动同步回 SQLite 工作区。
- 自动同步只在镜像内容真实变化后触发，避免无变化刷新时把旧镜像重复写回内部工作区。

### 2. 桌面端关闭即退出

- 标题栏关闭按钮现在执行应用退出。
- 系统窗口关闭请求也会退出应用。
- 原先“关闭后隐藏到托盘继续运行”的行为收敛为托盘菜单里的轻量模式入口。

### 3. 托盘轻量模式与 AI 状态

- 托盘右键菜单新增“进入轻量模式”，用于隐藏主窗口并保留后台执行能力。
- 托盘菜单会显示当前 AI 状态，例如空闲、AI 运行中、工具读写中、等待用户、上次失败。
- 左键点击托盘图标仍可显示主窗口。

### 4. 版本同步

- 项目版本更新到 `0.2.7`。
- 同步更新 `package.json`、`package-lock.json`、`Cargo.toml`、`Cargo.lock` 和 `tauri.conf.json`。
- README 的版本徽章、下载链接和更新日志入口已同步到 `0.2.7`。
- 更新检测相关测试改为以 `0.2.7 -> 0.2.8` 作为新版本场景。
